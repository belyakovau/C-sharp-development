# Task Manager - Консольный менеджер задач

Профессиональное консольное приложение для управления задачами (TODO List) на C#.

## 🎯 Описание

Task Manager — это полнофункциональное консольное приложение, которое позволяет:

- Создавать, читать, обновлять и удалять задачи (CRUD)
- Искать задачи по названию и описанию
- Фильтровать задачи по статусу выполнения
- Сортировать задачи по дате, приоритету или названию
- Автоматически сохранять данные в JSON файл
- Просматривать статистику по задачам

## 🛠️ Технологии

- **C#** (.NET 6.0+)
- **System.Text.Json** — для работы с JSON
- **LINQ** — для фильтрации, сортировки и поиска
- **ООП** — объектно-ориентированное программирование

## 📋 Требования

- .NET SDK 6.0 или выше
- Windows, Linux или macOS
